使用Myeclipse习惯后突然使用IDEA须知
发表于 2020-05-16| 更新于 2020-06-10|Java入门
阅读量:

一 、配置Idea

  • 配置JDK Maven

二、配置自动补齐

  • 找到KeyMap菜单

    File–》Settings–》KeyMap(快捷键ctrl+alt+s)进入快捷键设置界面。

  • 移除占用Alt+斜杠的快捷键

    需要将Basic的快捷键修改为Alt+/，但Alt+/被Cyclic Expand Word占用，所以先修改Cyclic Expand Word的快捷键，右键Remove Alt+斜杠。

  • 设置Basic快捷键为Alt+斜杠

    右键Add Keyboard Shutcut，然后在键盘上按下Alt+/,右键RemoveCTRL+空格。

二、必备快捷键

快捷键 功能
Ctrl+P 构造方法提示
Ctrl + Y 删除当前行
Ctrl + Alt +L 格式化代码
Ctrl +Shift+/ 注释当前行
Ctrl+Alt+V / Alt+Enter 对象补齐
psvm main()函数
sout system.out.println()
Ctrl+Shift+F 关键字全局搜索：搜索当前项目的文件、搜索maven中的class类
Ctrl+O 查看可实现（重写）的方法
Ctrl+H 查看继承树或者实现树
Ctrl+N 关键字搜索所有的类包括maven里的class
Ctrl + 加号 展开方法
Ctrl + 减号 收起方法
Ctrl +shift+ 加号 展开所有方法
Ctrl + shoft+减号 收起所有方法
iter/itar/itco/itli for 循环
