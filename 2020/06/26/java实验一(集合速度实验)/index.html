结果讨论

对于增删

因为LinkedList插入是先通过for循环遍历找到index然后直接插入，而ArrayList在指定位置添加或者删除元素主要耗时间的地方在与拷贝index后面元素，这个操作会移动index后面所有的元素。

对于遍历

毋庸置疑的说是选用Arraylist更快一点，因为通过源码我们可以知道get方法是直接返回index位置上的元素

而Linkedlist需要先遍历节点。
